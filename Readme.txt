				Описание работы приложения:

Программа банкомат.

Банкомат имеет две функции:
- Внесение наличных
- Снятие наличные

Вы можете выбрать номинал купюры для выдачи при выборе операции "снять наличные".

Также для наглядности банкомат отображает свой статус и свои настройки по умолчанию.

Если в банкомате закончились купюры или банкомат заполнен купюрами,в этом случае
банкомат перейдёт в статус "unavailable" и станет недоступен для дальнейшей работы.


			Программные средства необходимые для запуска и работы

1) IDE Visual Studio версия .NET Framework 4.7.2
2) NuGet package System.Windows.Interactivity.WPF
